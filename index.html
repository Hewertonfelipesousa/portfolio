Minha caixinha de ferramentas

Descubra as habilidades e ferramentas que domino e que me permitem criar soluções criativas e funcionais para meus clientes.

  • HTML

    HTML (HyperText Markup Language) é uma linguagem de marcação que define a estrutura e o significado do conteúdo de uma página web. É a base para a construção da maioria das páginas da internet.

  • CSS

    CSS (Cascading Style Sheets) é uma linguagem de estilo que define como os elementos de uma página web devem ser apresentados. O CSS é usado para estilizar a aparência de um site, alterando a fonte, cor, espaçamento, e mais.

  • JavaScript

    JavaScript (JS) é uma linguagem de programação usada para criar páginas web interativas. É uma das linguagens de programação mais populares do mundo.

  • Python

    Python é uma linguagem de programação de alto nível, interpretada e orientada a objetos. É uma das linguagens de programação mais populares e versáteis, usada em diversos campos, como ciência de dados, desenvolvimento web e machine learning.

  • PHP

    PHP (Hypertext Preprocessor) é uma linguagem de programação de código aberto, muito utilizada para desenvolvimento web. É compatível com a maioria dos sistemas operacionais e pode ser embutida em HTML.

  • React

    React é uma biblioteca JavaScript que permite criar interfaces de usuário (UI) para aplicações web e móveis. Foi criado pelo Facebook e é uma das ferramentas mais usadas para desenvolvimento web.

  • Gerenciamento de Rede

    Gerenciamento de rede é o processo de supervisionar, monitorar, configurar e manter uma rede de computadores. O objetivo é garantir que a rede esteja sempre disponível, segura e funcionando de forma eficiente.
